LATERAL COMPRESSION OR STRECH SETTINGS

- Stretch type: select between the two stretch types
- Piecewice: select to define the stretch as a piecewise function
- Sine: select to define the stretch as a sine wave
- Table: modify the piecewise times and the corresponding values
- Add a row button: add a row at the end of the table
- Remove a row button: remove a row from the end of the table
- Figure: visualization of the stretch function
- Load button: load a stretch function from a file
- Save button: save the current stretch function into a file
- Reset button: reset the stretch function back to the values before calling the dialog
- Default: restore the default piecewise function from the file
- Close: save changes and close the dialog
