OPTOGENETIC DISPLACEMENT ANALYSIS

- Start: select the displacement starting time point from the dropdown
- End: select the displacement ending time point from the dropdown
- Window size: size of the visualized area in the area plot or in the 1D plot along the junctions
- Max displacement: define the upper limit of the shown displacement range
- Simulation selection box: select the imported simulation to analyse
- Select cells (bottom): select cells two neighboring junction analysis
- Center to selection (bottom): center the area plot to the selected cells
- Save button (bottom): save the data from the current 1D plot to a csv file
- Close (bottom): close the dialog
