GRADIENT STIFFNESS SETTINGS

- Rotate gradient: define an angle to rotate the gradient clockwise in degrees
- Table: define the positions and the stiffness values at those positions
- Add a row button (+): add a row to the end
- Remove a row button (-): remove a row from the end
- Figure: plot of the gradient
- Load button: load gradient from a file
- Save button: save gradient to a file
- Reset button: reset the changes to the gradient before opening the dialog
- Default: reset the gradient to the default from a file
- Close: close the dialog
